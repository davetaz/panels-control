# PanelsD.CC Control Public Build Artifacts

This repository folder exists as a release reference point for **PanelsD.CC Control** packaged frontend assets.

The JavaScript in this folder is compiled and obfuscated for distribution and runtime use, so it is not intended to be a developer-facing source repository.

For platform architecture and component context (Design, Control, Connect), see:
https://panelsd.cc/architecture

## Purpose

- Provide a stable location for packaged `public` assets used in release workflows.
- Support build and packaging automation for PanelsD.CC Control.
- Keep historical/reproducible release outputs for reference.

## Source Code

Source code is maintained in the main PanelsD.CC project, not in this artifact folder.
